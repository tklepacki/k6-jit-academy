## Wymagania

### Narzędzia

Upewnij się, że Twój laptop nie ma żadnych ograniczeń, takich jak brak uprawnień do instalacji narzędzi lub ograniczony dostęp do sieci (VPN itp.).

Zainstaluj następujące oprogramowanie:

* **Visual Studio Code** - [https://code.visualstudio.com/download](https://code.visualstudio.com/download)

* **k6** - [https://grafana.com/docs/k6/latest/set-up/install-k6](https://grafana.com/docs/k6/latest/set-up/install-k6)

* **k6 Studio** - [https://grafana.com/docs/k6-studio/set-up/install/](https://grafana.com/docs/k6-studio/set-up/install/)

* **Docker**:

  * [https://docs.docker.com/desktop/setup/install/linux/](https://docs.docker.com/desktop/setup/install/linux/) - Linux
  * [https://docs.docker.com/desktop/setup/install/windows-install/](https://docs.docker.com/desktop/setup/install/windows-install/) - Windows
  * [https://docs.docker.com/desktop/setup/install/mac-install/](https://docs.docker.com/desktop/setup/install/mac-install/) - Mac

* Najnowszą wersję przeglądarki **Chrome** lub **Firefox**

* **GIT** - [https://git-scm.com/download/win](https://git-scm.com/download/win)

### Rejestracja kont:

* **Grafana Cloud k6** – utwórz darmowe konto: [https://grafana.com/products/cloud/k6/](https://grafana.com/products/cloud/k6/)

* **Ecommerce Store** – utwórz konto w Amazona: [https://ecommercestore.fly.dev/](https://ecommercestore.fly.dev/)

### Weryfikacja środowiska

* **k6**

  * Wpisz w konsoli: `k6` i upewnij się, że wyświetla się informacja o dostępnych komendach k6.

* **Docker**

  * Wpisz w konsoli: `docker -v` i upewnij się, że wyświetla się wersja Dockera.

* **Pobierz obrazy Dockera za pomocą poniższych poleceń:**

  * `docker pull grafana/grafana`
  * `docker pull influxdb:1.8`

* **Sklonuj repozytorium:**

  * Zrób fork repozytorium: [https://github.com/tklepacki/k6-jit-academy.git](https://github.com/tklepacki/k6-jit-academy.git) na swoje konto GitHub. Jeśli nie masz konta GitHub, najpierw je utwórz.
  * Sklonuj sforkowane repozytorium (ze swojego konta)
  * Przejdź do folderu repozytorium i upewnij się, że projekt został pobrany

* **Otwórz Visual Studio Code -> File -> Open Folder (wybierz folder ze sklonowanym projektem).**

  * Upewnij się, że wszystkie pliki projektu są widoczne w eksploratorze plików VSC.
